# Main options

| Option | Description  |
|-------------|-----|
| -s --src | Source directory of top-level _.try_ files |
| --dest -d | Destination directory for the generated _.html_ files |
| _--outfile -o_ |  Output file name |
| --filelist | list of top-level _.try_ files  |
